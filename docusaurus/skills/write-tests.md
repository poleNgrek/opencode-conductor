# write-tests

Skill for authoring unit and integration tests against existing code.

## What it does

- Detects the test framework.
- Identifies untested code paths.
- Drafts tests with proper mocking, edge cases, and assertions.
- Runs the new tests and reports pass/fail before write.

## Recommended permission

`ask` — writes new test files.
