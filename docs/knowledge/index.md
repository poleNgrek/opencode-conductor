Knowledge in this kit is dual-audience (agents and humans) and layered:

- Project and area durable knowledge in `AGENTS.md`
- Leaf-level knowledge by source-tree-mirror convention
- Branch/session narrative in `MERGE_REQUEST.md`, `LOG.md`, and optional `PHASES.md`

Primary contracts:

- `docs/PATH_CONTRACT.md` for path and schema invariants
- `commands/scaffold-knowledge.md` for creation/discovery modes
- `commands/project-knowledge-refresh.md` for proposal-first maintenance
